# 精弘毅行报名系统 · 前端

精弘毅行报名系统是使用Vue3 + TS + 微信相关接口开发的团队报名系统,并配套搭建了管理员小程序端用于点位签到打卡


## 技术栈

- Vue
- Naive UI
- Axios
- qrcode-vue

## 如何开始

> **配置`/walk-view/src/config/Server.ts`**

**本地运行测试**

```bash
yarn dev
```

**编译 realease 版本**

```bash
yarn build
```


## 注意事项

本项目依赖于微信中的浏览器,并且不支持热重载,代码修改效果只能上传至服务器部署后在微信浏览器中查看
